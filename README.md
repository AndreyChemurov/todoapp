# TODO-list app
Консольное приложение TODO-list с использованием postgres. В программе есть возможность создавать аккаунт (имя и пароль), вход и выход из аккаунта.
Все пользователи (username, password) хранятся в таблице users.
При создании нового пользователя создается таблица <username>_todos, в которую будут добавляться задания.
  <username>_todos состоит из 4 столбцов: id SERIAL PRIMARY KEY, task_name TEXT, date_and_time TIMESTAMP, comment TEXT.
Команды (для текущего пользователя):
    1. show all - выводит все имеющиеся задания.
    2. show today - выводит все имеющиеся сегодня задания.
    3. show by index <index> - выводит всю информацию о задании с индексом <index>.
    4. show by date <date> - выводит все имеющиеся задания по дате <date>.
    5. show by time <time> - выводит все имеющиеся сегодня задания по времени <time> (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
    6. show by task <task> - выводит все имющиеся задания с названием <task>.
    7. add task <task> <date> <time> [<comment>] - добавляет в таблицу задание с указанными параметрами.
    8. delete all - удаляет все имеющиеся задания из таблицы.
    9. delete today - удаляет все имеющиеся сегодня задания из таблицы.
    10. delete by index <index> - удаляет задание по указанному индексу <index>.
    11. delete by date <date> - удаляет задания по указанной дате <date>.
    12. delete by time <time> - удаляет все задания сегодня по указанному времени <time> (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
    13. delete by task <task> - удаляет все имющиеся задания с названием <task>.
    14. rewrite all - переписать все имеющиеся задания в таблице (изменить название, дату, время, комментарий).
    15. rewrite today - переписать все имеющиеся сегодня задания в таблице (изменить название, дату, время, комментарий).
    16. rewrite by index <index> - переписать задание по указанному индексу <index> (изменить название, дату, время, комментарий).
    17. rewrite by date <date> - переписать все все имеющиеся задания в таблице по заданной дате <date> (изменить название, дату, время, комментарий).
    18. rewrite by time <time> - переписать все имеющиеся сегодня задания в таблице по заданному времени <time> (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
    19. rewrite by task <task> - переписать все имеющиеся задания с заданным именем <task>.
