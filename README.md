# TODO-list app
Консольное приложение TODO-list с использованием postgres. В программе есть возможность создавать аккаунт (имя и пароль), вход и выход из аккаунта. <br />
Все пользователи (username, password) хранятся в таблице users. <br />
При создании нового пользователя создается таблица username_todos, в которую будут добавляться задания. <br />
username_todos состоит из 4 столбцов: id SERIAL PRIMARY KEY, task_name TEXT, date_and_time TIMESTAMP, comment TEXT. <br />
**Команды (для текущего пользователя):**
* show all - выводит все имеющиеся задания.
* show today - выводит все имеющиеся сегодня задания.
* show by index **index** - выводит всю информацию о задании с индексом **index**.
* show by date **date** - выводит все имеющиеся задания по дате **date**.
* show by time **time** - выводит все имеющиеся сегодня задания по времени **time** (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
* show by task **task** - выводит все имющиеся задания с названием **task**.
* add task **task date time** [**comment**] - добавляет в таблицу задание с указанными параметрами.
* delete all - удаляет все имеющиеся задания из таблицы.
* delete today - удаляет все имеющиеся сегодня задания из таблицы.
* delete by index **index** - удаляет задание по указанному индексу **index**.
* delete by date **date** - удаляет задания по указанной дате **date**.
* delete by time **time** - удаляет все задания сегодня по указанному времени **time** (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
* delete by task **task** - удаляет все имющиеся задания с названием **task**.
* rewrite all - переписать все имеющиеся задания в таблице (изменить название, дату, время, комментарий).
* rewrite today - переписать все имеющиеся сегодня задания в таблице (изменить название, дату, время, комментарий).
* rewrite by index **index** - переписать задание по указанному индексу **index** (изменить название, дату, время, комментарий).
* rewrite by date **date** - переписать все все имеющиеся задания в таблице по заданной дате **date** (изменить название, дату, время, комментарий).
* rewrite by time **time** - переписать все имеющиеся сегодня задания в таблице по заданному времени **time** (время может указываться в формате HH:MI, в таком случае будут выведены все задания с HH:MI:00 по HH:MI:59; либо может указываться в формате HH:MI:SS, тогда будут выведены все задания, которые соответствуют введенному времени).
* rewrite by task **task** - переписать все имеющиеся задания с заданным именем **task**.
